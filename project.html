EverWear

GithubLink
지속 가능한 패션 뷰티 브랜드 추천 웹 서비스
2인 팀에서 개발 파트(프론트엔드, 백엔드) 담당
EverWear는 온라인 편집샵 형태로 개인화 추천 및 유저 간 공유 기능을 제공하는 react, node.js 기반 프로젝트입니다. RESTful API를 생성하여 새롭게 등록되는 브랜드에 대한 정보 및 카테고리 분류가 빠르게 수행되도록 구현하였습니다. 브랜드의 상세 소개와 캠페인 홍보 페이지 구현하였고, 찜하기 기능과 마이페이지/관리자 페이지를 제공합니다.

What I did

레이아웃 및 UI/UX 디자인 프론트엔드 전체 개발 백엔드 전체 개발 데이터베이스 설계, 관리

Stack

React를 사용한 프론트엔드 기능 구현 Node.js를 사용한 백엔드 서버 개발 MongoDB를 사용한 유저와 포스트 데이터 관리 Redux를 사용한 유저 상태 관리 JWT를 통한 유저 인증 HOC를 활용한 권한별 라우팅 처리 multer library를 사용한 파일 저장 CSS Framework Ant Design 활용 React Hooks을 적용하여 코드 작성

전지적 지역 시점

Github Link
공공데이터 Open API를 활용한 전국지역정보알리미 웹
3인 팀에서 프론트엔드 개발, DB 설계 담당
전지적 지역 시점은 전국의 모든 국민이 언제 어디서나 교육 정보를 확인하고 각종 체육 시설 및 도서관을 이용할 수 있도록 도와주는 웹 서비스입니다. 기존 시도, 시 군 구별로 운영하던 서비스를 확대하여 전국 단위의 서비스를 제공합니다. 공공데이터포털의 Open API를 호출하여 전국에 산재되어 있는 시설 기관들의 정보를 시도별, 시 군 구별 접근하여 원하는 지역의 강의 정보를 얻고, 각종 시설의 위치 및 상세 정보를 확인할 수 있습니다.

What I did

데이터 전처리 프론트엔드 전체 기능 개발 DB 설계 및 SQL 쿼리문 작성

Stack

JSP, CSS를 활용한 프론트엔드 개발 jQuery로 Ajax 데이터 통신 Web Server 구축(Apach, tomcat) PostgreSQL기반 개발 트리거, 롤업 생성으로 데이터 상태 관리 자동화 작업 노션, 깃헙을 활용한 프로젝트 일정 관리
